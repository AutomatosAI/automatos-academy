## 🎬 Video Overview prompt (~8 min) — Prompts, context & few-shot that work
```
Audience: a professional studying for the GH-300 GitHub Copilot exam — capable but not an expert here.
Explain plainly first, then precisely. Cover ONLY: how to structure prompts, craft context, and use
zero-shot vs few-shot with Copilot — prompt structure and specificity, supplying context (open files,
selections, prompt files, and via MCP and Spaces), and iterative refinement. Map every point to what a
candidate must DECIDE on the exam: given a weak result, which prompt/context lever to pull. ~8 minutes:
open with why better prompts change the output (and it's tested), one worked example (a vague prompt
refined with context + a few-shot example into a good suggestion), the top 2–3 distractors (thinking a
longer prompt beats a well-scoped one; forgetting that open context/Spaces/MCP shape the answer;
zero-shot vs few-shot confusion), and a one-line "on the exam, remember…". Stay strictly grounded in
the provided sources; do not invent prompt techniques.
```
